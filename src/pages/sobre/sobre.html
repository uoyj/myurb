Como Funciona Privacidade Sobre o MyUrb
O MyUrb oferece serviços básicos, sendo que alguns são online e outros offline, para auxiliar o usuário através do transporte público de Curitiba. Serviços Online:
  • Busca de pontos próximos: na tela inicial, aperte o botão "PERTO DE MIM" para ser redirecionado para uma tela que apresente os pontos de ônibus que se encontrem perto de sua localização. Caso não esteja com o GPS ativado, é necessário inserir um endereço manualmente atráves da barra de pesquisa desta tela.
  • Gerar rotas de ônibus: insira um endereço de origem e outro para o destino, através das barras de pesquisas, e aperte o botão de "BUSCAR" para ser redirecionado para uma tela que apresente as rotas encontradas.
    • Serviços Offline:
  • Consultar itinerários: através do menu lateral esquerdo, selecione a opção "ITINERÁRIOS". Na nova tela, seleciona a linha de ônibus desejada para visualizar o itinerário realizado.
  • Consultar itinerários: através do menu lateral esquerdo, selecione a opção "HORÁRIOS". Na nova tela, seleciona a linha de ônibus desejada para visualizar os horários de partida do ponto inicial.
  • Consultar pontos: através do menu lateral esquerdo, selecione a opção "PONTOS". Na nova tela, seleciona o ponto de ônibus desejada para visualizar seus detalhes.
  • Código QR: através do menu lateral esquerdo, selecione a opção "CÓDIGO QR". Na nova tela, *digite* o código para visualizar detalhes sobre o ponto de ônibus.

    O MyUrb é uma aplicação que realiza a coleta de informações para aprimorar seu funcionamento. Aqui estão alguns esclarecimentos sobre esta prática. Quais informações são coletadas? São coletadas informações sobre as viagens realizadas (data, horário e ponto de ônibus de partida, horário e ponto de ônibus de chegada e linha utilizadas), adversidades reportadas sobre pontos e linhas de ônibus. Alguma informação pessoal é coletada? Não. Tudo ocorre de forma anônima, visando não prejudicar de nenhuma forma o usuário. Como as informações são utilizadas? As informações coletadas são utilizadas durante a criação de rotas, penalizando rotas que contém pontos e linhas de ônibus com maior quantidade de adversidades. É possível utilizar a aplicação sem fornecer nenhuma informação? Sim, e todas as funcionalidades estarão disponíveis. Para isto, durante a criação de rota, é necessário que ao final do percurso, a viagem seja cancelada. Quem pode acessar as informações? Qualquer pessoa pode visualizar através da página web do projeto, chamada MyUrbAnalytics.
    O projeto MyUrb surgiu como resultado de um Trabalho de Conclusão de Curso de Graduação em Engenharia de Computação na Universidade Tecnológica Federal do Paraná (UTFPR) - campus Curitiba. O principal objetivo do MyUrb é auxiliar os usuários através do transporte público de Curitiba, ao mesmo tempo que disponibiliza as informações anônimas coletadas para auxiliar a administração da cidade. O projeto possui código aberto e utiliza dados abertos da cidade de Curitiba para funcionar.
